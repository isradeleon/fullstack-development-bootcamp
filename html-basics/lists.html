Screen readers

List are important to give context about your page to the screen readers for blind people.

Each list item (li tag) can also contain other HTML elements (including block elements), to create a more complex structure:

  • An h4 title of the sublist:

    A pararaph describing this sublist.

    • Sub list item 1
    • Sub list item 2
    • Sub list item 3

  • Another title here

    More content

    • Sub list item 1
    • Sub list item 2
    • Sub list item 3
    A div tag contained within the list item tag.
